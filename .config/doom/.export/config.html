emacs server

Makes emacs startup faster! https://stackoverflow.com/questions/56933893/systemctl-invoke-an-emacs-server In terminal emulator systemctl –user enable emacs

Restart or

systemctl –user start emacs

Then emacs can be started in terminal mode with

emacsclient -nw

or in windowed mode with

emacsclient -c
